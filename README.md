# JoshuaAngelo Portfolio

A clean, minimal portfolio website designed with a dark GitHub-inspired theme.

## Files included

- `index.html` — portfolio homepage with your profile and experience
- `styles.css` — styling for the dark theme and layout

## View the portfolio by link

To make this site viewable by link, publish the repository with GitHub Pages.

1. Open the repository settings on GitHub.
2. In the Pages section, select the `main` branch and root folder.
3. Save the settings.

Once GitHub Pages is active, your portfolio link should be:

`https://yakshazxc.github.io/JoshuaAngelo.Portfolio./`

If the repository name changes, use the URL shown in the Pages settings.
